Problem

Chef has infinite coins in denominations of rupees 555 and rupees 101010.

Find the minimum number of coins Chef needs, to pay exactly XXX rupees. If it is impossible to pay XXX rupees in denominations of rupees 555 and 101010 only, print −1-1−1.

Input Format

  • First line will contain TTT, number of test cases. Then the test cases follow.
  • Each test case contains of a single integer XXX.

Output Format

For each test case, print a single integer - the minimum number of coins Chef needs, to pay exactly XXX rupees. If it is impossible to pay XXX rupees in denominations of rupees 555 and 101010 only, print −1-1−1.

Constraints

  • 1≤T≤10001 \leq T \leq 10001≤T≤1000
  • 1≤X≤10001 \leq X \leq 10001≤X≤1000

Subtasks

  • Subtask 1 (100 points): Original constraints.

Sample 1:

Input
Output
3
50
15
8

5
2
-1

Explanation:

Test Case 111: Chef would require at least 555 coins to pay 505050 rupees. All these coins would be of rupees 101010.

Test Case 222: Chef would require at least 222 coins to pay 151515 rupees. Out of these, 111 coin would be of rupees 101010 and 111 coin would be of rupees 555.

Test Case 333: Chef cannot pay exactly 888 rupees in denominations of rupees 555 and 101010 only.
